🧮 Multi-Theme Calculator



A sleek, fully functional web-based calculator that allows users to switch between five distinct modern UI design trends on the fly. Built entirely with HTML5 🌐, CSS3 🎨, and Vanilla JavaScript ⚡, this project serves as both a practical utility and a showcase of advanced frontend styling techniques.
📖 Overview
Tired of the standard, boring calculator interface? 🤔 The Multi-Theme Calculator offers a personalized experience by letting you choose the visual aesthetic that fits your mood! From raw, structural designs to soft, futuristic layouts, this project seamlessly transitions between completely different CSS architectures using a single underlying HTML and JavaScript logic structure. 🚀
✨ Key Features
🔄 Dynamic Theme Switching: Instantly toggle between five unique UI themes without reloading the page.
➕ Standard Arithmetic: Supports addition, subtraction, multiplication, division, and modulo operations.
📱 Responsive Design: A clean, centered layout that looks gorgeous and works perfectly on desktops, tablets, and mobile screens.
📦 Zero Dependencies: Built entirely with raw, vanilla web technologies—no heavy frameworks or external libraries required!
🎨 Themes Included
This calculator explores five of the most popular web design trends:
🔮 Glassmorphism: Features a frosted-glass effect with backdrop blurs, semi-transparent panels, and animated, glowing background blobs.
🧱 Neo Brutalism: A bold, unapologetic design characterized by stark contrasts, thick black borders, flat high-saturation colors, and sharp, boxy shadows.
⚪ Minimalism: A distraction-free, elegant layout relying on clean lines, ample whitespace, and ultra-subtle borders.
🌈 Colorfulism: A vibrant, playful, and energetic aesthetic utilizing bright gradients, heavily rounded corners, and soft, colored drop-shadows.
🔘 Neumorphism (Soft UI): Simulates physical tactile objects with a monochromatic color palette, utilizing clever inset and outset shadows to make buttons look extruded right from the background.
🛠️ Tech Stack
🌐 HTML5: Semantic structuring of the theme selector and calculator interface.
✨ CSS3: Advanced styling, CSS grid/flexbox, custom animations (@keyframes), backdrop-filters, and complex dual-source box-shadows.
⚡ Vanilla JavaScript: DOM manipulation, dynamic class toggling for theme management, and mathematical computation handling.
